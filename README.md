# Algorand NFT Mint Example

- Algorand SDK 
- PureStake API
- dotenv


## 1.`generate_algo_account.js` Creates a fundable account on Algorand

- Generates account 
- Generates mnemnonic 
- Generates account address



## 2. `generate_nft.js` Creates a single Algorand NFT/ASA via Javascript SDK / PureStake

- Generates ASA
